1. 공통 모티베이션

Cross-Attention은 WHERE 정보를 담고 있다

Cross-Attention의 역할

핵심 관찰: Diffusion Model의 Cross-Attention Map (CAM)은 "어떤 텍스트 토큰이 이미지의 어디에 영향을 주는지"를 나타낸다.
$\text{CAM}(h_t^l, c) = \text{softmax}\!\left(\frac{Q(h_t^l) \cdot K(c)^\top}{\sqrt{d}}\right)$

$Q = W_Q^l \cdot h_t^l \quad\text{(이미지 features)}, \quad K = W_K^l \cdot c \quad\text{(텍스트/개념 조건)}$
DAG “Cross-attention map으로 unsafe region을 감지(detect)하고, 그 영역만 guidance”
PbE “Cross-attention은 이미지 조건에서도 spatial control을 제공 — 텍스트 대신 이미지 임베딩 사용 가능”

공통 문제의식: 텍스트 조건의 한계

문제: "nude person"이라는 텍스트가 충분한가?

  • Background Leakage: CAM("nude person")이 배경까지 번진다
  • Specificity 부족: "nude"가 고양이, 벽에도 반응
  • 모호성: 텍스트만으로 fine-grained visual concept 표현 어려움

2. DAG: Guideline Token Optimization

Detect-and-Guide (Li et al., 2025)

DAG 핵심 아이디어 (1): Token Optimization

핵심: “nude person” 텍스트 대신, 소수의 annotated 이미지로 토큰 임베딩 $c^*$를 최적화하여 정확한 CAM 추출

CAM Loss (Pixel-level CE)

$\mathcal{L}_{\text{CAM}} = \mathbb{E}_{(i,y)\sim(\mathcal{X},\mathcal{Y})} \;\mathbb{E}_{t\sim\mathcal{T},\; l\sim\mathcal{L}} \;\frac{1}{K}\sum_{k}\sum_{h,w} \Big[-\text{OneHot}(y^{GT})_{h,w} \cdot \log\big(\hat{A}_{k,h,w}(h_t^l[i],\; c)\big)\Big]$

Optimization Objective

$c^* = \arg\min_c \;\mathbb{E}_{(i,y)} \;\mathcal{L}_{\text{CAM}}$
  • $\hat{A}$: Cross-attention map (min-max normalized)
  • $y^{GT}$: Ground truth mask (Grounded-SAM)
  • $\mathcal{L} = \{l \mid h_t^l.\text{size} \in [16^2, 32^2]\}$ (고해상도 layer만)
  • 5 unsafe + 3 safe images로 충분

결과

  • Background leakage 해소
  • Non-target concept (cat, wall) 무반응
  • $\hat{A}(c^*) \geq 0.5$ ⇒ unsafe region 감지
  • Unseen 샘플에 일반화
장점: 한 번 최적화하면 inference 시 추가 비용 $\approx$ 0

DAG 핵심 아이디어 (2): Adaptive Safety Guidance

Safety Guidance 수식 (DAG)

$\tilde{\epsilon}_\theta = (1-s_g)\,\epsilon^\varnothing + s_g\Big(\epsilon^p - \mathbf{S}_{c_s} \cdot \mathbf{M}_{c_s} \cdot \epsilon^{c_s}\Big)$

Scale Map $\mathbf{S}_{c_s}$

$\mathbf{S}_{c_s} = s_c \cdot \text{Area}_{0.5}(\hat{A}) \;\odot\; \mathcal{T}_{0.01}(\hat{A})$
  • $s_c = \frac{5}{H \times W}$: base weight
  • $\text{Area}_{0.5}(\hat{A})$: unsafe 영역 크기 (CAM $\geq$ 0.5인 픽셀 수)
  • $\mathcal{T}_{0.01}(\hat{A})$: magnitude rescaling ($0.01 \leq a \leq 1 \to [1, 5]$)

Mask $\mathbf{M}_{c_s}$

$\mathbf{M}_{c_s} = \mathbb{I}[\hat{A} \geq 0.01] \;\odot\; M_{cs}^{\text{SEGA/SLD}}$
  • $\hat{A} \geq 0.01$: potentially unsafe region (gate)
  • 기존 SEGA/SLD mask와 결합
  • $t_{cs} = 5$: warm-up 5 step만 (SLD는 7-15)
핵심 차별점: CAM magnitude가 클수록 더 강한 guidance → adaptive strength + spatial precision

3. Paint by Example: Information Bottleneck

Yang et al., CVPR 2023

PbE 핵심 아이디어 (1): Information Bottleneck

문제: Self-supervised 학습에서 reference image를 그대로 copy-paste하는 trivial solution에 빠진다

Naive Solution (실패)

$c = \text{CLIP}_{\text{all}}(x_r)$
(class token + 256 patch tokens = 257 tokens)
  • Spatial 정보가 그대로 유지
  • 모델이 단순히 copy-paste 학습
  • Test 시 reference와 다른 이미지 ⇒ 실패

Information Bottleneck (성공)

$c = \text{MLP}\Big(\text{CLIP}_{\text{class}}(x_r)\Big)$
(class token만 사용 = 1×768 vector)
  • $224 \times 224 \times 3 \to 1024$차원 벡터로 압축
  • High-freq detail 제거 → semantic 정보만 보존
  • 모델이 의미를 이해하고 재생성하도록 강제
↓
Key Insight: CLIP class token은 이미지의 semantic essence를 담지만 spatial layout은 버린다.
이것이 바로 concept-level probe에 필요한 속성이다.

PbE 핵심 아이디어 (2): Augmentation + CFG

Strong Augmentation

$c = \text{MLP}\Big(\text{CLIP}\big(\mathcal{A}(x_r)\big)\Big), \quad \bar{m} = 1 - \mathcal{D}(m)$
  • $\mathcal{A}$: flip, rotation, blur, elastic transform → train-test gap 축소
  • $\mathcal{D}$: irregular mask distortion (Bezier curve) → 실제 편집 상황 모방

Classifier-Free Guidance로 유사도 제어

$\tilde{\epsilon}_\theta(y_t, c) = \epsilon_\theta(y_t, v) + s\Big(\epsilon_\theta(y_t, c) - \epsilon_\theta(y_t, v)\Big)$
  • $v$: learnable unconditional vector (학습 중 20% dropout)
  • $s$: guidance scale — 클수록 reference image에 가까운 생성

4. 공통점과 차이점

DAG vs Paint by Example — 수렴하는 통찰, 다른 접근

공통점: 수렴하는 핵심 통찰

1. 텍스트보다 이미지가 낫다

  • DAG “non-optimized 'nude person' 텍스트는 background leakage + specificity 부족”
  • PbE “A picture is worth a thousand words — 텍스트 설명은 fine-grained control 부족”

2. Cross-Attention의 Semantic Entanglement 활용

  • DAG attribute(nude) + object(human) 결합이 CAM에서 자연스럽게 entangle
  • PbE CLIP class token이 cross-attention을 통해 spatial control 제공

3. 고해상도 Layer가 정밀한 Spatial 정보를 갖는다

  • DAG $\mathcal{L} = \{l \mid \text{size} \in [16^2, 32^2]\}$ — 64×64는 too noisy
  • PbE 동일한 U-Net cross-attention 구조 활용 (SD 기반)

차이점: 접근 방식의 분기

관점 DAG Detect-and-Guide PbE Paint by Example
목적 Safe generation (unsafe concept 제거) Exemplar-based image editing
조건 소스 최적화된 토큰 임베딩 $c^*$ CLIP class token of reference image
학습 필요 Token optimization (5+3 images, GT mask) Full model fine-tuning (64 V100, 7일)
CAM 활용 Detection mask → spatial guidance 암묵적 (cross-attention이 자동 align)

Inference 비용 +1 extra K projection (negligible) Single forward (standard diffusion)
일반화 Unseen prompts에 일반화 (검증됨) Arbitrary reference images에 일반화

5. v13: Training-Free Exemplar Probe

두 논문의 장점을 결합한 우리의 접근

v13의 핵심 질문

DAG의 정밀한 CAM detection을 PbE의 training-free information bottleneck으로 달성할 수 있는가?
↓
DAG에서 가져온 것
  • Cross-attention probing으로 spatial detection
  • 고해상도 layer 선택 ($16^2, 32^2$)
  • Detection map → adaptive guidance
PbE에서 가져온 것
  • CLIP class token as information bottleneck
  • 이미지 기반 조건: “a picture is worth 1000 words”
  • Training-free projection 가능성
v13의 새로운 기여
  • CLIP exemplar → SD text space projection
  • Zero-shot probe (학습 0)
  • CAS gating과의 통합

Step 1: Exemplar Embedding (Offline)

$\bar{f}_{\text{target}} = \frac{1}{N}\sum_{i=1}^{N} \frac{\text{CLIP}_{\text{cls}}(x_i^{\text{nude}})}{||\text{CLIP}_{\text{cls}}(x_i^{\text{nude}})||}$
  • N=16개 exemplar의 CLIP class token 평균
  • $\bar{f} \in \mathbb{R}^{768}$: PbE의 information bottleneck
  • BOS + $n$ copies + EOS + PAD → $[1, 77, 768]$

Step 2: Pre-compute Probe Keys (Offline)

$K_{\text{target}}^l = W_K^l \cdot c^*_{\text{target}}, \quad \forall l \in \mathcal{L}$
  • $\mathcal{L} = \{l \mid \text{res} \in [16, 32]\}$: DAG와 동일한 layer 선택
  • One-time cost: 16개 linear projection
  • Inference 중 고정 (frozen)

v13 수식: WHEN (Global CAS)

WHEN: 이 prompt가 unsafe concept을 포함하는가? — CAS로 판단
$\text{CAS}(t) = \cos\big(d_{\text{prompt}},\; d_{\text{target}}\big) = \frac{(\epsilon^p - \epsilon^\varnothing) \cdot (\epsilon^{\text{tgt}} - \epsilon^\varnothing)}{||\epsilon^p - \epsilon^\varnothing|| \;\cdot\; ||\epsilon^{\text{tgt}} - \epsilon^\varnothing||}$
$\text{should\_guide}(t) = \big[\text{CAS}(t) > \tau\big] \;\wedge\; \big[t \in \text{window}\big]$
$\tau = 0.6$ (default), sticky mode: 한 번 trigger되면 유지
DAG와의 차이: DAG는 $\hat{A}(c^*) \geq 0.5$를 global detection에 사용하고 warm-up step($t_{cs}=5$)으로 on/off 제어. v13은 noise space에서 CAS를 직접 계산하여 더 빠르고 robust한 triggering.

v13 수식: WHERE (Exemplar Cross-Attention Probe)

WHERE: 이미지의 어디가 unsafe한가? — Cross-attention probe로 감지

Probe Attention (During Forward Pass)

$\hat{A}^l = \text{softmax}\!\left(\frac{Q(h_t^l) \cdot K_{\text{target}}^{l\top}}{\sqrt{d}}\right) \in \mathbb{R}^{H \times S \times K}$

$h_t^l$: eps_prompt forward의 hidden state (cached), $K_{\text{target}}^l$: pre-computed exemplar key

Spatial Mask Aggregation

$A_{\text{spatial}}^l = \max_{k \in \text{tokens}} \;\frac{1}{H}\sum_h \hat{A}^l_{h,:,k} \quad\text{(head avg, token max)}$

$A_{\text{final}} = \text{normalize}\!\left[\frac{1}{|\mathcal{L}|}\sum_{l \in \mathcal{L}} \text{interpolate}(A_{\text{spatial}}^l, 64\times64)\right]$
cf. DAG: $c^*$는 pixel-level CE loss로 최적화 → background leakage 명시적 제거. v13: CLIP class token의 information bottleneck이 암묵적으로 leakage를 줄임 (training 불필요).

v13 수식: Soft Mask Construction

$M_{\text{soft}} = \text{GaussianBlur}_{5\times5}\!\Big[\sigma\!\big(\alpha \cdot (A_{\text{final}} - \theta_s)\big)\Big]$

$\alpha = 10$ (sigmoid sharpness), $\theta_s = 0.3$ (spatial threshold), blur $\sigma = 1.0$

v13 수식: HOW (Safety Guidance)

HOW: mask가 주어졌을 때 어떻게 안전하게 guidance하는가?

Hybrid Mode (Recommended)

$\epsilon_{\text{final}} = \epsilon_{\text{cfg}} - s_t \cdot M_{\text{soft}} \cdot (\epsilon^{\text{tgt}} - \epsilon^\varnothing) + s_a \cdot M_{\text{soft}} \cdot (\epsilon^{\text{anc}} - \epsilon^\varnothing)$

Hybrid Projection Mode

$d_{\text{safe}} = d_{\text{prompt}} - p_s \cdot \text{proj}_{d_{\text{target}}}(d_{\text{prompt}}), \quad \text{proj}_{d_t}(d_p) = \frac{d_p \cdot d_t}{||d_t||^2} d_t$

$\epsilon_{\text{safe}} = \epsilon^\varnothing + s_g \cdot d_{\text{safe}}$

$\epsilon_{\text{final}} = \epsilon_{\text{safe}} \cdot (1 - s_a \cdot M_{\text{soft}}) + \epsilon_{\text{anc\_cfg}} \cdot (s_a \cdot M_{\text{soft}})$
cf. DAG: $\tilde{\epsilon} = (1-s_g)\epsilon^\varnothing + s_g(\epsilon^p - \mathbf{S}_{c_s} \cdot \mathbf{M}_{c_s} \cdot \epsilon^{c_s})$ — DAG는 safety prompt $c_s$로부터 $\epsilon^{c_s}$ 계산. v13은 target/anchor 분리로 더 세밀한 제어.

핵심 메시지

DAG는 “정확한 detection을 위해 학습이 필요하다”고 했고,
PbE는 “이미지 조건의 semantic bottleneck이 핵심”이라고 했다.

v13은 PbE의 CLIP information bottleneck을
DAG의 cross-attention probing framework에 대입하여
학습 없이도 정밀한 spatial unsafe detection을 달성한다.
v13의 장점
  • 완전 training-free → 새 concept에 즉시 적용
  • GT mask annotation 불필요
  • CLIP shared space 덕분에 text space projection 자연스러움
  • CAS gating으로 benign prompt FP 최소화
남은 과제
  • DAG의 optimized $c^*$보다 detection 정밀도가 낮을 수 있음
  • Background leakage 완전 해소는 미검증
  • Exemplar 이미지 선택의 영향 분석 필요
  • Hard-to-erase nudity (memorized images) 한계
