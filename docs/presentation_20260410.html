Headline: EBSG achieves SOTA safety rates across 7 I2P categories while preserving image quality (FID ~3.7), outperforming both fine-tuning and training-free baselines.

Motivation

Problem: T2I Safety Gap

T2I Models Generate Harmful Content
Nudity, violence, hate imagery, self-harm from adversarial prompts
Existing Solutions
  • Fine-tuning (ESD, RECE): costly per-concept training, catastrophic forgetting
  • SLD/SEGA: global suppression, quality degradation
  • SAFREE: token projection, limited concept coverage
Our Approach: EBSG
  • No training / fine-tuning needed
  • Selective intervention: only when & where needed
  • Example-based: add new concepts in <1 min
  • Multi-concept simultaneous erasure
  • 7 I2P categories with single framework

Key Insight: Use exemplar embeddings to detect when a concept aligns with the denoising direction, where it localizes via cross-attention + CLIP similarity, and how to steer it toward a safe anchor — all at inference time.

WHEN: Concept Alignment Score

$\text{CAS}(t) = \cos(\epsilon_\text{prompt} - \epsilon_\varnothing, \; \epsilon_\text{target} - \epsilon_\varnothing) \quad \xrightarrow{\text{if } > \tau} \text{activate}$
  • Measures alignment between prompt direction and target concept at each step $t$
  • Only activates guidance when CAS > threshold $\tau$ (0.4–0.6)
  • Sticky CAS: once triggered, stays active for remaining steps
Result: Benign prompts (COCO) pass through without any intervention → FID ~3.7, CLIP Δ < 0.1

WHERE: Dual-Probe Localization

Text Probe

Cross-attention maps for target keywords
(“nude”, “violence”, etc.)
$M_\text{text} = \sigma(\alpha \cdot (\max_k A_{w_k} - \theta))$
Catches explicit keyword triggers

Image Probe (CLIP Exemplar)

Compare U-Net features with
pre-computed CLIP exemplar embeddings
$M_\text{img} = \sigma(\alpha \cdot (\max_n \cos(f, e_n) - \theta))$
Catches adversarial/implicit triggers
Fusion: $M = \min(1, M_\text{text} + M_\text{img})$ — union of both probes for robustness

HOW: Safe Guidance Modes

Anchor Inpainting

$\hat\epsilon_\text{safe} = \hat\epsilon_t - s \cdot M \cdot (\epsilon_\text{target} - \epsilon_\varnothing)$
Subtract unsafe concept signal within masked region. Best for text-explicit concepts.

Hybrid Guidance

$\hat\epsilon_\text{safe} = \hat\epsilon_t + s_t \cdot M \cdot (\epsilon_\text{anchor} - \epsilon_\text{target})$
Push toward safe anchor direction. Best for visually implicit concepts (nudity).
