Method Recap

Example-Based Spatial Guidance (EBSG)

When · Where · How

WHEN
Global CAS Gate
→
WHERE
Per-family
Dual Probe
→
HOW
Anchor Inpaint
or Hybrid
WHEN
$c_t = \cos(\varepsilon^p_t-\varepsilon^\varnothing_t,\ \varepsilon^{tgt}_t-\varepsilon^\varnothing_t)$
Trigger if $c_t > \tau_{cas}$ (sticky).
WHERE
Per-family $M_f = \max(M^{img}_f,\ M^{txt}_f)$
Pixel-wise winner-take-all across $F{=}4$ families.
HOW
anchor: $\varepsilon^{safe}\!\leftarrow(1{-}\beta_f)\varepsilon^{safe}+\beta_f\tilde\varepsilon^{anc}_f$
hybrid: $\varepsilon^{safe}\!\leftarrow\varepsilon^{safe}-sM_f(\varepsilon^{tgt}_f-\varepsilon^{anc}_f)$
One artifact = exemplar pack. Each pack ships $F{=}4$ family-specific target/anchor exemplars (text + CLIP image features). New concept = drop a new pack, no retraining.

Evaluation Protocol

Why we adopt VLM-based 4-class judging

Why a 4-Class VLM Rubric, Not NudeNet / Q16

Scalar safety detectors hide collapse.
  • NudeNet / Q16 ask one binary question: is this image harmful?
  • Blurred / noisy / semantically destroyed images $\to$ counted as safe
  • A method that destroys the image scores the same as one that removes only the unsafe concept
Our rubric (Qwen3-VL-8B).
  • 4-class label per image: $\{$Safe, Partial, Full, NotPeople/NotRelevant$\}$
  • Semantic collapse $\to$ NotPeople, explicitly not safe
  • Canonical $\mathrm{SR} = (\textit{Safe}+\textit{Partial})\,/\,\textit{Total}$ — no strict/loose split
  • Validated on MJ-Bench: 94.7% without-tie accuracy, on par with GPT-4o / Gemini
Honesty boost (planned). A small human-majority study ($\sim$20 annotators × 100 I2P images per concept, 800 images total) will back the evaluator-choice claim in the appendix.
Bibliographic note. Our prior workshop paper (archival) used a related VLM-based protocol. To stay safe under double-blind we do not self-cite it; the protocol is introduced fresh in the experiments section, with the same canonical SR formula.

Paper Narrative — Re-tightened This Week

Two confounds, one fix.
  1. Method side: baselines collapse the concept into a single direction and intervene globally → semantic drift on benign content.
  2. Eval side: scalar safety detectors miss semantic collapse → can't measure semantic-preserving erasure.
Our framing. Family-indexed exemplar pack + 4-class VLM rubric address both at once.

Summary & Talking Points

  • Anchor vs. hybrid is a tradeoff, not a choice. 3 concepts each — we report both as first-class and let the deployer pick by anchor cleanliness.
  • Hybrid sweeps nudity (UD 98.6, RAB 97.4, MMA 84.7, P4DN 97.4); anchor closes the gap on harassment / illegal / self-harm.
  • One pack, three backbones. Average SR on MJA: SD 1.4 83.7, SD 3 74.8, FLUX.1 92.5.
  • Probe = both beats single channels on every I2P concept.
  • Evaluator: 4-class VLM rubric with explicit collapse class — no scalar-detector loophole; ongoing human study.
  • Open: multi-concept image-probe capacity, missing baseline cells, COCO-FID for hybrid.
Bottom line. The When–Where–How decomposition with a small, user-editable exemplar pack gives consistent SoTA among training-free methods, transfers to MM-DiT / DiT backbones unchanged, and the remaining items are scope expansions, not method blockers.
