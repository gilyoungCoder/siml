Problem Setting

Two confounds, one fix

Two Confounds in T2I Safety Today

Method-side: monolithic suppression.
  • Existing safeguards collapse a multifaceted concept into a single negative direction.
  • Suppression is applied globally across every spatial location and timestep.
  • Inevitable semantic drift on the benign portion of the prompt.
  • Both training-based (ESD/SDD/RECE) and training-free (SLD/SAFREE) share this bottleneck.
Eval-side: scalar detectors hide collapse.
  • NudeNet / Q16 answer one binary question: is this image harmful?
  • Blurred / noisy / collapsed images $\to$ counted as safe.
  • A method that destroys the image scores the same as one that removes only the unsafe concept.
  • No way to measure semantic-preserving erasure honestly.
EBSG addresses both at once. A family-indexed exemplar pack replaces the single direction with localized, sub-concept-aware steering; a 4-class VLM rubric flags semantic collapse explicitly so it can’t masquerade as safety.

One concept-specific artifact: the exemplar pack. Adding or recombining concepts means dropping in family directories of (target keyword, target image, anchor keyword)—no fine-tuning, no closed-form weight edit.

Family-Indexed Exemplar Pack

Decompose a broad concept $h$ into $F$ semantically disjoint families, each with $K$ paired entries.

$\mathcal{P}_{h} = \big\{\,(w_{h,f,k}^{\mathrm{harm}},\ e_{h,f,k}^{\mathrm{harm}},\ w_{h,f,k}^{\mathrm{safe}})\,\big\}_{f\in[F],\, k\in[K]}$
Family-level pooling for steering. Per family, average $K$ phrasings in CLIP space: $\bar{w}_{h,f}^{\mathrm{harm}}, \bar{e}_{h,f}^{\mathrm{harm}}, \bar{w}_{h,f}^{\mathrm{safe}}$.
Phrasing nuisance cancels; shared family semantics reinforce.
Concept-level pooling only for detection. A single broader aggregate $\bar{w}_{h}^{\mathrm{harm}}$ drives the When gate—binary trigger, no need for finer granularity.

Anchor: calm interaction, peaceful scene

WHEN — Concept Alignment Score

$\mathrm{CAS}_{h}(t) \;=\; \cos\!\big(\bm{d}_{c}(t),\ \bm{d}_{h}(t)\big),\qquad \bm{d}_{c}(t)=\bm{\epsilon}_{\theta}(z_{t},c)-\bm{\epsilon}_{\theta}(z_{t},\varnothing)$
  • One scalar per timestep. Cosine of the prompt’s CFG difference vector against the concept-level harmful direction.
  • Sticky activation. Once $\mathrm{CAS}_{h}(t){>}\tau$ at any $t_{0}$, latch on for all $t{>}t_{0}$ to prevent flicker.
  • Concept-level on purpose. Detection is binary; family-level granularity is reserved for Where/How.
  • Cost: one extra UNet forward per step, independent of $F$. Parameterization-invariant (transfers to velocity / flow predictors).
Why this design helps benign prompts. $\mathrm{CAS}_{h}$ aligns weakly with $\bm{d}_{h}$ on safe inputs and the trajectory passes through the base sampler unchanged— the explanation for why COCO FID stays at 6.31 while four nudity benchmarks all clear 84%+ SR.

WHERE — Dual Probe + Winner-Take-All

Text probe: $M^{\mathrm{txt}}_{h,f}(i,j)=\sigma\!\big(\alpha\,(A^{\mathrm{txt}}_{h,f}(i,j)-\theta)\big)$
Image probe: same form, but conditioning tokens replaced with family CLIP image embeddings.
Within-family union: $\bar{M}_{h,f} = \max(M^{\mathrm{txt}}_{h,f},\ M^{\mathrm{img}}_{h,f})$
Across families: $M_{h,f}(i,j)=\mathbf{1}\!\big[f=\arg\max_{f'} \bar{M}_{h,f'}(i,j)\big]\cdot\bar{M}_{h,f}(i,j)$
Two failure modes covered by one partition.
  • Explicit erasing: prompt carries unsafe keywords (“naked”, “blood”) → text probe dominates.
  • Implicit erasing: metaphorical prompt evades keywords (“a dark exchange in an alley”) → image probe dominates.
  • Union+WTA: pixels are routed to a single winning family.
Outcome. Pixel-wise disjoint partition $\{M_{h,f}\}_{f=1}^{F}$ — a region is steered toward exactly one family’s anchor, with no double-counting.

HOW — Mask-Gated target → anchor (EBSG)

$\hat{\bm{\epsilon}}^{h\text{-safe}}_{c,t} \;=\; \hat{\bm{\epsilon}}_{c,t} \;+\; s_{h}\sum_{f\in[F]}\Big[\,M_{h,f}\odot\big(\bm{d}_{h,f}^{\mathrm{safe}}(t)-\bm{d}_{h,f}^{\mathrm{harm}}(t)\big)\Big]$
  • Family-level steering, not concept-level. Each family pushes toward its own anchor (e.g., exposed parts $\to$ clothed; weapon $\to$ calm interaction).
  • Active only inside the masked region. $M_{h,f}{=}0$ leaves the base sampler untouched on benign content.
  • Towards a safe alternative, not just away from harm. $\bm{d}^{\mathrm{safe}}_{h,f}-\bm{d}^{\mathrm{harm}}_{h,f}$ is a directed redirect, not a blunt subtraction.
Single update equation. One mask-gated target→anchor formula handles every concept; no separate “anchor-inpaint” mode, no per-concept mode-flip table, no $\beta$ blend. Per-concept tuning lives entirely in $(s_{h},\, \tau_{h},\, \theta)$ and the contents of $\mathcal{P}_{h}$.

Multi-Concept Erasure: One Inference, $C$ Concepts

Active set: $\mathcal{H}_{t}=\{(h,f):\mathrm{CAS}_{h}(t){>}\tau_{h}\}$.
Cross-concept WTA: $M_{h,f}(i,j)=\mathbf{1}\!\big[(h,f)=\arg\max_{(h',f')\in\mathcal{H}_{t}}\bar{M}_{h',f'}(i,j)\big]\cdot\bar{M}_{h,f}(i,j)$.
$\hat{\bm{\epsilon}}^{\,\mathrm{safe}}_{c,t}=\hat{\bm{\epsilon}}_{c,t}+\sum_{(h,f)\in\mathcal{H}_{t}} s_{h}\big[M_{h,f}\odot(\bm{d}^{\mathrm{safe}}_{h,f}-\bm{d}^{\mathrm{harm}}_{h,f})\big]$
  • Per-concept CAS gates run independently. Only concepts that actually fire on the prompt enter the active set.
  • Single cross-concept WTA over all active (concept, family) pairs. At most one $(h,f)$ contributes to each pixel; cross-concept overlaps are resolved structurally.
  • Per-concept knobs are independent: each $h$ owns its own $\tau_{h}$, $s_{h}$, and probe thresholds; the update equation itself is shared.
Why SAFREE-multi collapses but EBSG-multi doesn’t. SAFREE projects every concept against a shared toxic subspace at the token level, so 6 concepts shrink the safe subspace until generation collapses to noise. EBSG keeps masks pixel-wise disjoint and gates each concept independently—benign content stays benign even when 6 packs are loaded simultaneously.

4-Class VLM Rubric

Why scalar detectors are the wrong measurement

VLM-Based Evaluation Protocol (Qwen3-VL-8B)

Scalar detectors hide collapse.
  • NudeNet / Q16: single binary “harmful?” question.
  • Blurred / noisy / collapsed image $\to$ counted as safe.
  • Cannot distinguish destruction from concept removal.
  • Empirical confound documented in our appendix.
4-class rubric. $\{\textsc{Safe},\ \textsc{Partial},\ \textsc{Full},\ \textsc{NotRelevant}\}$
  • NotRelevant flags semantic collapse explicitly.
  • $\mathrm{SR}=(\textsc{Safe}+\textsc{Partial})/\mathrm{Total}$—canonical, no strict/loose split.
  • Validated on MJ-Bench: 94.7% w/o-tie (on par with Gemini Ultra 95.1).
  • Cross-checked against a human-majority annotation study (appendix).
Bibliographic note. Our prior workshop paper (archival) used a related VLM-based protocol; to stay safe under double-blind we re-introduce the rubric fresh in the experiments section without self-citation, using the same canonical SR formula.

Image-Count Saturation — $K{=}4$ is Enough

SR vs. number of image exemplars per family $K \in \{1, 2, 4, 8, 16\}$ (SD v1.4, I2P top-60). Dashed line: chosen default $K{=}4$.

Sexual saturates at $K{=}1$. ~90% SR with a single image per family; adding more does not help (and slightly hurts at $K{=}16$). Anchor-cleanliness: crisp visual signature, easy to localize.
Violence / hate plateau by $K{=}4$. Visually heterogeneous (blood / weapon / brutal action; symbols / text / hateful imagery) so a few exemplars are needed, but the curve flattens by 4—no return on $K{=}8$ or $K{=}16$.
Why this matters for deployment. The exemplar pack stays small (4 images × 4 families per concept = 16 images), so adding a new concept is a directory commit, not a data-collection project. CLIP embeddings are precomputed once and the cost is negligible.

SD v1.4 across four I2P concepts. Columns: SR (Safe$+$Partial)\,$\uparrow$, Full violation\,$\downarrow$, NotRelevant (collapse)\,$\downarrow$. Methods: EBSG, SAFREE, SD v1.4 baseline.
