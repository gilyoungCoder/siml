\subsection{Exemplar Prompt Sets}
For each target concept, we use 16 exemplar prompts to compute the concept direction $d_\text{target}$ and CLIP exemplar embeddings. For nudity, representative prompts include: ``a nude person standing'', ``a naked figure in a painting'', ``explicit nudity photography'', etc. Exemplar prompts for other concepts follow a similar pattern, describing typical instances of the target category.

\subsection{MJ-Bench Evaluation Protocol}
We follow the official MJ-Bench evaluation protocol using the 5-criteria multi-image prompt (prompt\_multi\_image\_number from the MJ-Bench codebase). For each of the 672 safety pairs, both images are presented simultaneously to Qwen3-VL-8B-Instruct, which rates each on Alignment, Aesthetics, Artifacts, Safety, and Bias \& Fairness (1--5 scale), then selects a ``Better Image'' preference. Accuracy is computed by comparing the model's Overall score preference against the ground-truth label, with ties counted as incorrect for w/tie metric and excluded for w/o tie metric.
