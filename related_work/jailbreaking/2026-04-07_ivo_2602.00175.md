# The Illusion of Forgetting: Attack Unlearned Diffusion via Initial Latent Variable Optimization (arXiv:2602.00175)

- Title: **The Illusion of Forgetting: Attack Unlearned Diffusion via Initial Latent Variable Optimization**
- Authors: Manyi Li, Yufan Liu, Lai Jiang, Bing Li, Yuming Li, Weiming Hu
- Date checked: 2026-04-07
- Primary sources:
  - arXiv abstract: https://arxiv.org/abs/2602.00175
  - arXiv PDF: https://arxiv.org/pdf/2602.00175.pdf

---

## 1. 한 줄 요약
이 논문은 unlearning-based diffusion defense가 사실 concept를 완전히 지운 게 아니라 **언어 심볼과 내부 지식 사이의 mapping만 부분적으로 망가뜨린 것**이라고 주장하고, 그 잔존 기억(dormant memory)을 **initial latent optimization** 으로 다시 깨우는 공격 **IVO**를 제안한다. 세 논문 중에서 가장 명확하게 **prompt가 아니라 latent 쪽**으로 무게중심이 이동한 논문이다.

## 2. 문제의식
논문이 겨냥하는 건 일반 safety checker 우회가 아니다.
핵심 타깃은 **unlearned diffusion model** 이다.

저자들의 큰 주장:
1. unlearning은 concept knowledge 자체를 완전히 삭제하지 못한다.
2. 대부분은 symbol-to-knowledge mapping을 부분적으로 끊었을 뿐이다.
3. 따라서 적절한 latent trigger를 찾으면, 끊긴 mapping을 다시 이어 unsafe concept를 재활성화할 수 있다.

즉 이 논문은 기존 jailbreak 문헌보다 더 공격적으로,
**“forgetting is an illusion”** 이라는 해석을 전면에 둔다.

---

## 3. 핵심 개념: dormant memory
저자들은 이미지 생성에서 특정 concept가 나오는 과정을
- linguistic symbol
- underlying knowledge
사이의 mapping으로 본다.

unlearning이 진짜 성공했다면,
그 concept에 해당하는 knowledge 자체가 사라져야 한다.
그런데 prompt-based attack에서도 여전히 non-negligible ASR이 나온다는 건,
knowledge는 남아 있다는 신호라고 본다.

논문은 이를 **dormant memories** 라고 부른다.
즉,
- 지식은 남아 있고
- 접근 경로만 일부 망가졌으며
- latent를 잘 조절하면 다시 활성화 가능하다는 주장이다.

---

## 4. distribution gap 해석
이 논문이 흥미로운 부분은, unlearned model과 standard model 사이의 denoising distribution 차이를
**unlearning strength의 지표**로 해석한다는 점이다.

- distribution discrepancy가 크면 더 강하게 지운 것처럼 보임
- discrepancy가 작으면 더 많은 valid pathway가 남아 있음
- 그리고 이 gap을 줄이면 dormant memory 재활성화가 쉬워짐

즉 단순히 “공격 잘 된다/안 된다”가 아니라,
**distributional discrepancy 자체를 defense quality indicator로 읽는다**.

이건 prompt-jailbreak 논문보다 훨씬 내부 메커니즘 해석에 가깝다.

---

## 5. 왜 prompt가 아니라 latent인가?
저자들은 prompt-based attack의 한계를 분명하게 지적한다.

- vocabulary-level perturbation
- trainable prefix
- prompt engineering
이런 것들은 결국 semantic space 안에서 비슷한 concept를 찾는 방식인데,
unlearning 과정에서 관련 concept의 mapping도 같이 훼손되기 때문에,
강력한 unlearned model 상대로는 한계가 크다고 본다.

그래서 IVO는 질문을 바꾼다.

> 왜 굳이 prompt를 최적화하지? 왜 initial latent를 직접 최적화하지?

논문의 답:
- latent space는 더 넓고 연속적이며
- unlearning 영향이 prompt보다 덜 직접적이고
- dormant memory를 깨울 수 있는 경로가 더 많이 남아 있을 수 있다.

---

## 6. IVO 방법
IVO는 세 단계로 구성된다.

### 6.1 Image Inversion
- NSFW 이미지를 DDIM inversion으로 latent space에 보낸다.
- 즉 random noise에서 시작하지 않고, **이미 unsafe 방향성이 강한 시작점**을 확보한다.

논문은 이게 중요한 이유를 설명한다.
random latent는 너무 넓은 공간의 “안전한 지역”에서 출발할 가능성이 높다.
반면 inversion된 latent는 target unsafe memory의 주변 suboptimal region에서 시작하므로,
최적화가 훨씬 쉬워진다.

### 6.2 Adversarial Optimization
논문은 두 가지 loss를 결합해 latent를 다듬는다.
- **DML (distribution matching loss)**: unlearned DM의 noise trajectory를 standard DM 쪽으로 맞추려는 성격
- **DCL (direction calibration loss)**: 생성 결과가 NSFW semantics 쪽으로 가도록 조정

요지는,
- 한 loss는 “원래 unsafe state에 가까운 denoising dynamics”를 회복하고,
- 다른 loss는 “목표 unsafe 의미”를 유지하게 만드는 것.

### 6.3 Reused Attack
- 한 번 잘 된 latent를 pool에 저장
- 이후 공격에서 재사용

이건 꽤 실용적이다.
즉 IVO는 매번 비싼 optimization을 처음부터 다 하지 않고,
성공한 latent를 재활용해 공격 성공률과 효율을 높인다.

## 7. 핵심 결과
### 7.1 평균 성능
Table 1 (NSFW-High 50) 기준 평균:
- Sneaky: ASR **59.5**
- MMA: ASR **27.8**
- Ring: ASR **28.3**
- UDiff: ASR **54.0**
- **IVO: ASR 79.8**

FID/KID 측면에서도 IVO가 전반적으로 더 좋거나 경쟁력 있다.
즉 단순히 성공률만 높고 이미지 품질이 나쁜 게 아니라,
semantic consistency도 어느 정도 유지한다는 게 논문 메시지다.

### 7.2 개별 unlearning method들
예를 들어 Table 1에서:
- **ESD**: IVO ASR **98.0**
- **FMN**: IVO ASR **100.0**
- **SPM**: IVO ASR **100.0**
- **UCE**: IVO ASR **100.0**
- **AdvU**: IVO ASR **100.0**

반면 RECE, STEREO 같은 더 강한 경우에는 더 낮아지지만,
그래도 baseline 대비 상당히 강하다.

### 7.3 논문 전체 메시지
저자들은 본문에서
- “most unlearned DMs에서 over 70% ASR”
- “black-box scenario로도 extension 가능”
을 강조한다.

즉 IVO는 단순한 특수 케이스 공격이 아니라,
**unlearning defense 전반의 구조적 약점을 드러내는 stress test** 로 제시된다.

## 8. 강점
### 8.1 정말로 latent 관점이다
사용자가 말한 “latent 관점의 시도”라는 표현이 정확하다.
이 논문은 prompt wording이나 evaluator loop보다,
**denoising trajectory / initial latent / distribution alignment** 를 핵심 공격 surface로 본다.

### 8.2 unlearning을 메커니즘적으로 해석
많은 공격 논문은 우회만 보여준다.
이 논문은 거기서 더 나아가,
왜 우회가 가능한지를
- knowledge retention
- broken mapping
- distribution discrepancy
관점으로 설명하려 한다.

### 8.3 prompt-space ceiling을 넘으려는 시도
prompt-level attack이 막히는 상황에서,
latent-level trigger를 쓰는 건 분명 의미 있는 확장이다.
특히 unlearning robustness를 진짜 시험하려면,
이런 내부 상태 조작이 훨씬 본질적인 probe일 수 있다.

---

## 9. 약점 / 비판 포인트
### 9.1 일반 commercial API jailbreak와는 결이 다름
RPG-RT나 JPA는 black-box API red-teaming에 더 직접적이다.
반면 IVO는 기본적으로 **다운로드 가능한 unlearned model의 white-box 평가**에 가깝다.
물론 black-box extension도 다루지만, 논문 중심은 거기가 아니다.

즉 “상용 서비스 jailbreak의 mainstream”이라는 관점에서는,
IVO는 메인스트림이라기보다 **특정 defense setting(unlearning)에 대한 공격 연구**라고 보는 게 더 맞다.

### 9.2 시작점 의존성
DDIM inversion으로 얻은 시작점이 강력한 만큼,
이 방식은 어느 정도 **좋은 seed / 좋은 inversion sample**에 의존하는 측면이 있다.
실전 운영에서 데이터와 seed pool 품질이 성능에 영향을 줄 수 있다.

### 9.3 공격 모델이 더 강해짐
latent를 직접 최적화하는 순간,
실제 위협 모델은 prompt jailbreak보다 훨씬 강해진다.
그래서 방어 비교가 엄격하다는 장점도 있지만,
상용 API 유저 수준 공격과는 threat model이 다를 수 있다.

---

## 10. 우리 연구 맥락에서의 해석
이 논문은 사용자의 관찰을 이렇게 수정하게 만든다.

- **일반적인 jailbreak / commercial API red-teaming** 에서는 prompt optimization + evaluator loop가 주류에 가깝다.
- 하지만 **unlearning robustness를 공격하는 문맥** 에서는 latent optimization이 훨씬 강력한 대안이 될 수 있다.

즉,
> “diffusion jailbreak는 거의 다 prompt 최적화 아닌가?”
라는 질문에 대해,
**보통은 맞지만, unlearned diffusion을 깨는 쪽에서는 latent-space attack이 매우 강한 별도 축으로 등장한다**
라고 정리하는 게 좋다.

---

## 11. 다른 두 논문과의 관계
### vs JPA
- JPA: embedding-guided prefix prompt optimization
- IVO: initial latent optimization
- JPA는 CLIP/text 쪽에서 concept direction을 잡음
- IVO는 denoising/latent dynamics 쪽에서 dormant memory를 깨움

### vs RPG-RT
- RPG-RT는 black-box commercial API adaptive red-teaming
- IVO는 unlearning defense의 내부적 취약점 분석
- RPG-RT는 feedback loop가 핵심
- IVO는 distribution matching + latent refinement가 핵심

---

## 12. 최종 메모
세 논문 중 IVO는 가장 이질적이다.
왜냐하면 이 논문은 단순히 “더 좋은 jailbreak prompt”가 아니라,
**unlearning이 정말 concept를 지웠는가?** 라는 더 근본적인 질문을 던지기 때문이다.

그래서 이 논문은 diffusion jailbreak 일반론보다는,
**unlearning defense의 실질적 망각 여부를 검증하는 공격 논문**으로 읽는 편이 더 정확하다.
