## 1. 핵심 문제 정의

기존 concept erasure 방법들의 두 가지 근본적 한계를 지적:

1. **불완전한 제거 (Incomplete Erasure)**: UCE 등 기존 closed-form 방법으로 concept을 지워도, adversarial embedding을 유도하면 여전히 unsafe 이미지를 생성할 수 있음. 즉, "nudity"라는 단어의 직접 매핑만 끊었을 뿐, 모델 내부에 nudity 표현 능력이 남아있음.
2. **생성 능력 훼손**: 기존 fine-tuning 기반 방법(ESD, SA, CA)은 대량의 파라미터를 수정하여 무관한 concept의 생성 품질까지 떨어뜨림.

**핵심 질문**: UCE처럼 빠르면서도(3초), red-teaming attack에도 robust한 concept erasure가 가능한가?

## 2. 핵심 방법론

### 2.1 기반: UCE (Unified Concept Editing) 리뷰

UCE는 cross-attention의 K/V projection matrix를 closed-form으로 수정하는 방법:

- **목표**: erase할 concept `c_i`의 출력을 destination `c_i*` (e.g., empty text "")로 매핑, preserve할 concept `c_j`는 원래 출력 유지
- **수식**:

```
min_W  Σ_{c_i∈E} ||W·c_i - W^old·c_i*||² + λ₁·Σ_{c_j∈P} ||W·c_j - W^old·c_j||² + λ₂·||W - W^old||²_F
```

- **Closed-form solution** (Eq. 3):

```
W = W^old · (Σ c_i*·c_i^T + λ₁·Σ c_j·c_j^T + λ₂·I) · (Σ c_i·c_i^T + λ₁·Σ c_j·c_j^T + λ₂·I)^{-1}
```

- UCE는 K/V matrix만 수정 (U-Net 파라미터의 2.23%)하여 매우 빠름 (~1.2초)
- **한계**: erasure가 불완전 -- adversarial embedding으로 여전히 nude 이미지 생성 가능

### 2.2 RECE의 핵심 아이디어: Iterative Derive-and-Erase

RECE는 **adversarial fine-tuning의 아이디어**를 차용하되, 모두 closed-form으로 수행:

**반복 루프**:
1. **Embedding Derivation**: edited model 내에서 erased concept을 재생성할 수 있는 새로운 embedding `c'`를 찾음
2. **Model Editing**: 찾아낸 `c'`를 다시 erase
3. 1-2를 T epochs 반복

### 2.3 Embedding Derivation (새 임베딩 유도)

edited model(`W^new`)에서 original model(`W^old`)의 nudity 출력을 재현하는 embedding `c'`를 찾음:

**목적함수** (Eq. 4):
```
min_{c'} Σ_i ||W_i^new · c' - W_i^old · c||²
```

여기서 `W_i`는 U-Net의 각 cross-attention layer의 K/V projection matrix.

**Closed-form solution** (Eq. 5):
```
c' = (Σ_i W_i^{newT} · W_i^new)^{-1} · (Σ_i W_i^{newT} · W_i^old) · c
```

- `c'`는 edited model에 넣었을 때 original model에서 `c`를 넣은 것과 동일한 cross-attention 출력을 만드는 "adversarial embedding"
- 이것은 convex optimization이므로 unique global minimum을 가짐

### 2.4 Regularization Term

단순히 `c'`를 유도하면 모델의 일반 생성 능력이 훼손됨. 이를 방지하기 위한 정규화:

**정규화된 목적함수** (Eq. 7):
```
min_{c'} Σ_i ||W_i^new · c' - W_i^old · c||² + λ·||c'||²
```

**Regularized closed-form solution** (Eq. 8):
```
c' = (λI + Σ_i W_i^{newT} · W_i^new)^{-1} · (Σ_i W_i^{newT} · W_i^old) · c
```

- λ가 클수록 `c'`가 0에 가까워짐 → 모델 성능 보존
- λ가 작을수록 `c'`가 더 정확한 adversarial embedding → erasure 효과 극대화
- **이론적 근거 (Theorem 1)**: `c'`가 0으로 설정되면 W^new2와 W^new1의 차이가 최소화됨 → 무관한 concept에 대한 영향 최소

### 2.5 전체 알고리즘 (Algorithm 1)

```
Input: U-Net θ, erase set E, preserve set P, epochs T
1. θ' ← θ
2. W^old ← K/V matrices from θ
3. W^new ← UCE erasure (Eq. 3) on E
4. for t = 1 to T:
   a. for each c_i ∈ E:
      - Derive c_i' using W^new (Eq. 8, regularized)
   b. Erase all c_i' from W^new using UCE (Eq. 3)
   c. Update θ' K/V matrices with W^new
5. return θ'
```

- **실험 설정**: T=3 epochs for unsafe concepts, λ=1e-1 for Eq. 7, λ₁=λ₂=0.1 for Eq. 2
- **총 소요 시간**: ~3.4초 (RTX 3090 기준)

## 3. Training-Free 여부

**Training-free가 아님.** 다만 "fine-tuning-free"로 분류할 수 있음.

- 모델 파라미터(K/V projection matrix)를 **영구적으로 수정**함
- 단, gradient-based optimization/backpropagation을 사용하지 않음
- 모든 수정이 **closed-form** (행렬 연산)으로 이루어져 3초 내에 완료
- **Inference-time guidance가 아님** -- 모델 자체를 바꿈

| 분류 | 상태 |
|------|------|
| Fine-tuning required | X (closed-form) |
| Model weights modified | O (K/V projection matrices) |
| Training-free (inference-time) | X |
| Gradient-based | X |

## 4. WHEN / WHERE / HOW 프레임워크 분석

### WHEN (언제 guidance를 적용하는가)

- **적용 시점**: Inference 전, 모델 editing 단계 (pre-processing)
- **모든 timestep에 영향**: K/V matrix가 영구 수정되므로, 모든 denoising step에서 자동으로 적용됨
- **Timestep-selective가 아님**: CAS처럼 특정 timestep에서만 개입하는 것이 아니라, 모델 자체가 변경되어 항상 적용

### WHERE (어디에 적용하는가)

- **Cross-attention K/V projection matrices**: U-Net 내 모든 cross-attention layer의 W_k와 W_v
- U-Net 전체 파라미터의 **2.23%**만 수정
- **Spatial selectivity 없음**: 이미지의 특정 영역이 아닌, 전체 cross-attention 출력에 영향
- Text embedding → cross-attention 매핑 자체를 변경하므로, concept에 대한 전역적(global) 개입

### HOW (어떻게 개입하는가)

- **Closed-form K/V matrix editing**: UCE의 closed-form solution으로 projection matrix 직접 수정
- **Iterative adversarial derive-and-erase**: 
  1. Edited model에서 원본 concept을 재현하는 adversarial embedding 유도 (closed-form)
  2. 해당 embedding을 다시 erase (closed-form)
  3. 반복하여 concept representation을 철저히 제거
- **Regularization**: L2 norm으로 derived embedding을 0에 가깝게 유지 → 무관한 concept 보존

## 5. 실험 결과 요약

### 5.1 Nudity Removal (I2P dataset, 4703 prompts)

| Method | Total Nude Parts ↓ | CLIP ↑ | FID ↓ |
|--------|-------------------|--------|-------|
| SD v1.4 | 646 | **31.33** | - |
| ESD-u | 121 | 30.45 | 3.73 |
| UCE | 186 | 31.26 | **1.82** |
| SLD-Med | 354 | 30.95 | 2.62 |
| SA | 163 | 30.57 | 17.34 |
| CA | 95 | 31.16 | 7.87 |
| **RECE** | **66** | 30.95 | 2.82 |

- RECE가 nude body parts 수에서 최고 성능 (66개, 2위 CA는 95개)
- FID는 UCE(1.82)에 이어 2위(2.82) -- 생성 품질 잘 보존
- CLIP score도 합리적 범위 유지

### 5.2 Red-Teaming Robustness (ASR ↓)

| Attack | ESD-u | UCE | SLD-Max | SA | CA | RECE |
|--------|-------|-----|---------|----|----|------|
| UnlearnDiff | 66.20 | 79.58 | 82.39 | 77.46 | **65.49** | **65.49** |
| P4D | 63.38 | 80.28 | 77.46 | 78.87 | **60.56** | 64.79 |
| Ring-A-Bell | 69.72 | 33.10 | 66.20 | 22.54 | 25.35 | **13.38** |
| **Average** | 66.43 | 64.32 | 75.35 | 59.62 | 50.47 | **47.89** |

- RECE가 평균 ASR에서 최고 robustness (47.89%)
- 특히 Ring-A-Bell (black-box attack)에서 압도적: 13.38% vs 2위 SA 22.54%

### 5.3 Editing Duration

| Method | Params Modified (%) | Duration (s) |
|--------|-------------------|-------------|
| ESD | 94.65 | 3720 |
| UCE | 2.23 | 1.2 |
| SA | 94.65 | - (매우 느림) |
| CA | 100 | 1400 |
| **RECE** | 2.23 | 3.4 |

- UCE와 동일하게 2.23%만 수정, 3.4초에 완료

## 6. 장점

1. **극도의 효율성**: 3.4초 만에 concept erasure 완료 (closed-form, no gradient)
2. **강력한 robustness**: iterative derive-and-erase로 red-teaming attack에 대한 저항력 대폭 향상
3. **생성 품질 보존**: 2.23% 파라미터만 수정 + regularization term으로 FID 2.82 달성
4. **이론적 정당성**: regularization term의 효과에 대한 수학적 증명 제공 (Theorem 1, Appendix B)
5. **범용성**: nudity, artistic style, object class 등 다양한 concept에 적용 가능
6. **UCE 대비 명확한 개선**: 같은 파라미터 수정량으로 훨씬 나은 erasure + robustness

## 7. 한계

1. **모델 수정 방식**: inference-time method가 아니므로, 모델 weights가 영구 변경됨 → 원본 모델 유지 불가 (별도 저장 필요)
2. **Gender bias**: nudity erasure 시 female nudity에 편향 (SD 자체의 bias 상속) -- 논문에서도 인정
3. **Incomplete erasure 가능성**: 3 epochs 후에도 일부 adversarial embedding이 남아있을 수 있음 (epoch 수를 늘리면 생성 품질 훼손)
4. **Text-level 개입만**: cross-attention의 text→visual 매핑만 수정. Latent space에서의 concept representation은 건드리지 않음
5. **Multi-concept scalability**: 여러 concept을 동시에 지울 때의 간섭 효과 분석 부족
6. **Spatial awareness 없음**: 이미지의 특정 부분만 선택적으로 개입하는 기능 없음 → "옷을 입은 사람" 같은 benign case에서도 일부 영향 가능
7. **SDXL/DiT 등 최신 아키텍처 확장성**: SD v1.4 기반으로만 실험, 최신 모델 적용 불확실

## 8. 우리 방법(CAS + Spatial CFG)과의 관계/차이점

### 8.1 근본적 접근 차이

| 측면 | RECE | CAS + Spatial CFG (Ours) |
|------|------|-------------------------|
| **접근** | Model editing (weights 수정) | Inference-time guidance (weights 불변) |
| **Training** | Closed-form (3초, gradient 없음) | Training-free (완전 training-free) |
| **적용 시점** | 모델 배포 전 1회 | 매 inference마다 실시간 |
| **모델 변경** | K/V matrix 영구 수정 | 원본 모델 그대로 유지 |
| **되돌림** | 원본 weights 별도 저장 필요 | 자동 (guidance 끄면 원본) |

### 8.2 WHEN 비교

- **RECE**: Pre-deployment. 모든 timestep에 암묵적으로 적용 (weights가 바뀌었으므로)
- **Ours (CAS)**: Per-timestep adaptive. CAS score로 각 timestep에서 harmful 여부를 판단하여 선택적 개입 → 불필요한 개입 최소화, FP 감소

### 8.3 WHERE 비교

- **RECE**: 전체 cross-attention K/V matrix (global, spatial selectivity 없음)
- **Ours (Spatial CFG)**: Cross-attention map 기반으로 이미지 내 특정 spatial region에만 선택적 개입 → "사람이 있는 부분"에만 nudity guidance 적용 가능

### 8.4 HOW 비교

- **RECE**: K/V projection matrix의 closed-form 수정 (concept → destination 매핑 변경)
- **Ours**: SLD-style safe CFG: `epsilon_safe = epsilon_cfg - s * mask * (epsilon_target - epsilon_null)` → noise prediction level에서 concept 방향 제거

### 8.5 상호 보완 가능성

1. **RECE는 model-level, Ours는 inference-level**: 두 방법을 결합하면 이중 방어 가능
   - RECE로 모델의 기본적인 nudity 표현 능력 약화
   - CAS + Spatial CFG로 남은 edge case를 inference time에 처리
2. **RECE의 derived embedding 아이디어 차용**: RECE가 adversarial embedding을 찾는 방식을 우리 CAS threshold 설정에 참고 가능 -- "어떤 embedding이 harmful concept과 가까운가"의 기준
3. **우리 방법의 장점이 RECE의 한계를 보완**:
   - RECE의 gender bias → 우리의 spatial selectivity로 완화 가능
   - RECE의 모델 영구 수정 → 우리는 원본 유지
   - RECE의 전역적 개입 → 우리의 spatial mask로 정밀 개입

### 8.6 실험 비교 시 고려사항

- RECE는 I2P 전체(4703 prompts)에서 평가, 우리는 4개 nudity dataset(Ring-A-Bell, MMA, P4DN, UnlearnDiff) + I2P subset으로 평가
- RECE의 NudeNet threshold 0.6은 우리와 동일
- RECE는 COCO-30k로 benign evaluation (CLIP + FID), 우리는 COCO로 FP check + VQAScore
- **직접 비교 가능 지표**: NudeNet body part counts, COCO FID

### 8.7 논문 인용 시 포지셔닝

RECE는 "efficient model editing" 계열의 SOTA로, 우리 방법과는 **orthogonal한 접근**:
- RECE = **model-centric** (배포 전 모델 자체를 안전하게)
- Ours = **inference-centric** (추론 시 실시간으로 안전하게)
- Related work에서 "model editing approaches"로 분류하고, 우리 방법이 "training-free inference guidance"로서 complementary함을 강조
