## 핵심 문제 정의

기존 training-free negative guidance 방법들 (Shielded Diffusion/SPELL, Safe Denoiser)은 unsafe 콘텐츠를 피하기 위해 denoising 과정에서 repulsive force를 적용하지만, 두 가지 근본적 한계가 있었다:

1. **이론적 통합 프레임워크 부재**: SPELL과 Safe Denoiser가 서로 다른 heuristic에 기반하여 설계되었으며, 이들을 통합하는 확률론적 프레임워크가 없었다.
2. **Critical time window의 원리적 정당화 부재**: negative guidance를 *언제* 강하게 적용하고 *언제* 약하게/중단해야 하는지에 대한 수학적 근거가 없었다. Safe Denoiser는 경험적으로 early step에서만 적용하고, SPELL도 early step에서 자연히 강해지는 경향이 있었지만, 왜 그래야 하는지 증명하지 못했다.

**SGF가 해결하는 것**: MMD(Maximum Mean Discrepancy) potential을 사용한 energy-based negative guidance 프레임워크를 제안하여 (1) 기존 두 방법을 통합적으로 설명하고, (2) control-barrier function 이론으로 critical time window의 존재와 early stopping의 필요성을 수학적으로 증명한다.

## 핵심 방법론

### 4.1 MMD Potential 기반 Energy Formulation

unsafe 데이터셋 D- = {y_i}_{i=1}^N 가 주어졌을 때, 현재 샘플 x_t와 D- 사이의 (biased) squared MMD estimator를 potential function으로 정의:

$$E(x_t) \equiv \widehat{\text{MMD}}^2_{k_\sigma}(\{x_t\}, D^-)$$

여기서 k_sigma는 RBF kernel: k_sigma(x, y) = exp(-||x - y||^2 / (2 sigma^2))

이 potential의 gradient를 flow matching ODE에 추가:

$$\dot{x}_t = f_\theta(x_t, t) + \lambda(t) \nabla_x E(x_t)$$

gradient 계산 결과 (핵심 수식, Eq. 7):

$$\nabla_{x_t} \widehat{\text{MMD}}^2_{k_\sigma}(x_t, D^-) = \frac{2}{\sigma^2} Z(x_t) \left[ x_t - \sum_{i=1}^N w_i(x_t) y_i \right]$$

여기서:
- Z(x_t) = (1/N) * sum_i k(x_t, y_i): kernel 평균 (proximity 측정)
- w_i(x_t) = k(x_t, y_i) / (N * Z(x_t)): 각 negative sample의 가중치

**직관**: x_t가 unsafe sample y_i에 가까울수록 k(x_t, y_i)가 크고, Z(x_t)가 커져서 repulsive force가 강해진다. Force 방향은 x_t에서 가중 평균 위치로부터 멀어지는 방향이다.

### 4.2 기존 방법 통합 (Proposition 1 & 2)

**Proposition 1 (Safe Denoiser 복원)**: RBF kernel의 MMD gradient guidance는 Safe Denoiser의 weighted kernel repellency field과 (positive scalar 곱 차이로) 동일하다. 즉 Safe Denoiser는 MMD gradient의 특수한 경우이다.

**Proposition 2 (SPELL 복원)**: 적절한 bandwidth sigma를 선택하면, MMD gradient의 Gaussian force가 SPELL의 radial thresholded force F_rad(d) = alpha(r - ||d||)_+ * d/||d||과 특정 거리 d_0에서 magnitude가 일치한다. 즉 SPELL은 radius-thresholded MMD guidance의 인스턴스이다.

### 4.3 Critical Window 이론 (Theorem 2)

Control-barrier function 분석을 적용하여 핵심 결과를 증명:

**Forward-time dynamics** (s in [0, 1]):
$$\frac{dx}{ds} = \tilde{f}(s, x) + \beta(s) \nabla_x E(x)$$

**Theorem 2 (Forward-time critical window)**: barrier function h가 있고, Assumption 1 (boundary layer에서 gradient alignment + base drift bound)이 성립하면:

$$e^{\int_0^{s_c} L(\tau) d\tau} h(x_0) + \mu \bar{I}_L(s_c) \geq \delta$$

일 때, h(x_{s_c}) >= delta (deadline s_c까지 safe region에 도달).

**핵심 해석**:
- guidance budget (integral of beta)이 고정되면, **일찍 적용할수록** sufficient bound가 더 커진다 (w_L(u)가 u에 대해 감소하므로)
- 즉 **"earlier is better"** for safety guidance
- Critical window [0, s_c] 이후에는 beta = 0으로 설정해도 (early stop) 안전성이 유지되면서 fidelity가 향상된다

### 4.4 실용적 알고리즘 (Algorithm 1)

```
for t = T to 0:
    x_hat_0|t = E[x_0 | x_t]  (clean sample prediction, Tweedie formula)
    if t in C (critical window):
        x'_0|t = x_hat_0|t + lambda(t) * grad_x E(x_hat_0|t, D-)
    else:
        x'_0|t = x_hat_0|t  (no guidance)
    x_{t-1} = Solver(x_t, t, x'_0|t)
```

**구현 핵심 사항**:
- Negative guidance는 **x_0 space** (predicted clean sample space)에서 적용된다
- Bandwidth sigma는 adaptive하게 설정: top-k nearest neighbor distance 기반 heuristic (k=3)
- Critical window: diffusion의 경우 reverse time [1.0, 0.78] (early 22% steps), DDPM index 780~1000에 해당
- N=1 (단일 이미지 생성)이므로 GPU 병렬화가 자연스러움

## Training-free 여부

**완전한 Training-free 방법이다.**

- 모델 weight 수정 없음
- 추가 학습 없음
- Inference time에 MMD gradient를 denoising trajectory에 추가하는 방식
- 필요한 것: unsafe reference dataset D- (negative datapoints)만 있으면 됨
- 실험에서는 I2P에서 nude probability > 0.6인 515개 이미지를 D-로 사용

## WHEN / WHERE / HOW 프레임워크 분석

### WHEN (언제 guidance를 적용하는가)

**Critical window 이론에 기반한 early-step 적용**

- Control-barrier function theorem으로 수학적으로 정당화: guidance는 denoising 초기에 강하게, 이후에는 약하거나 꺼야 한다
- 실험적 최적 window: reverse time t in [1.0, 0.78] (DDPM 1000-step 기준 index 780~1000)
- **Ablation 결과 (Figure 4)**: [1.0, 0.8]이나 [1.0, 0.6] window가 최적, 나중 step으로 이동하면 ASR 급증
- Early stopping이 핵심: 전체 step에 guidance 적용하면 이미지 품질이 크게 저하 (FID 악화, diversity 감소)

**우리 방법(CAS)과 비교**: CAS는 매 step마다 cos(d_prompt, d_target)을 계산하여 threshold 기반으로 적용 여부를 **동적으로** 결정한다. SGF는 고정된 time window를 사전에 정의하여 적용한다. CAS가 더 adaptive하고 prompt-dependent한 WHEN 결정을 한다.

### WHERE (어디에 적용하는가)

**x_0 prediction space (clean sample space) 전체에 global하게 적용**

- Spatial selectivity가 전혀 없다
- 전체 latent/pixel space에 균일하게 repulsive force 적용
- Latent의 특정 영역이나 cross-attention map 기반의 공간 선택 없음

**우리 방법과 비교**: CAS + Spatial CFG는 cross-attention map이나 noise difference 기반으로 WHERE를 결정한다. 즉 unsafe 콘텐츠가 발생하는 spatial 영역에만 선택적으로 개입한다. SGF는 WHERE 차원이 완전히 부재하다.

### HOW (어떻게 개입하는가)

**MMD potential gradient 기반 repulsive force**

핵심 수식:
$$x'_{0|t} = \hat{x}_{0|t} + \lambda(t) \cdot \nabla_{\hat{x}_{0|t}} E(\hat{x}_{0|t}, D^-)$$

- E는 predicted clean sample과 unsafe reference set D- 간의 squared MMD
- Gradient가 repulsive force 역할: x_0 prediction을 D-로부터 밀어낸다
- lambda(t)는 guidance strength schedule (critical window 내에서 constant 또는 decaying)
- RBF kernel bandwidth sigma는 adaptive top-k heuristic으로 자동 설정

**우리 방법과 비교**: 우리는 SLD-style safe CFG를 사용한다:
- epsilon_safe = epsilon_cfg - s * mask * (epsilon_target - epsilon_null)
- Noise space에서 concept direction을 빼는 방식
- SGF는 x_0 space에서 data-driven repulsive force를 사용하고, 우리는 noise space에서 concept direction을 빼는 방식이다

## 장점

1. **이론적 통합**: SPELL과 Safe Denoiser를 MMD potential 프레임워크 아래 통합. 기존 heuristic 방법들에 확률론적 정당화 제공
2. **Critical window의 수학적 증명**: Control-barrier function으로 "early guidance is better"를 rigorous하게 증명. WHEN에 대한 원리적 답변
3. **Plug-and-play**: SLD, SAFREE 등 기존 training-free 방법과 자연스럽게 결합 가능 (negative guidance component만 교체)
4. **강력한 safety 성능**: SAFREE + SGF 조합이 SOTA ASR 달성
5. **낮은 computational overhead**: 기존 대비 추가 비용이 미미
6. **다양한 응용**: nudity 방지, diversity 향상, memorization 완화에 모두 적용 가능
7. **Diffusion + Flow Matching 모두 지원**: 이론이 두 프레임워크 모두에 적용됨

## 한계

1. **Spatial selectivity 부재**: 전체 이미지에 global하게 적용하므로, safe한 영역까지 불필요하게 영향받을 수 있다. Benign prompt에서의 품질 저하 가능성 (COCO FID 증가 경향)
2. **Reference data 의존**: unsafe reference dataset D-가 필요하다. 515개 이미지를 사전에 확보해야 하며, concept마다 별도의 D- 구성이 필요
3. **Prompt-agnostic WHEN 결정**: time window가 고정 ([1.0, 0.78])이므로, 모든 prompt에 동일하게 적용. Safe prompt에도 불필요하게 guidance가 적용될 수 있다
4. **Concept-specific하지 않음**: unsafe의 "어떤" concept을 지울지 구분하지 않고, D- 전체에 대해 repulsion 적용. Multi-concept erasing에서 fine-grained control이 어려움
5. **Boundary layer assumption**: 이론적 증명이 gradient alignment assumption에 의존. 실제 high-dimensional latent space에서 이 가정이 얼마나 잘 성립하는지 불명확
6. **COCO FID 열화**: Table 1에서 SAFREE + SGF의 COCO FID가 23.73으로, 순수 SAFREE (25.29) 대비 나아보이지만 CLIP이 30.36으로 약간 하락

## 우리 방법 (CAS + Spatial CFG)과의 관계/차이점

### 근본적 설계 철학 차이

| 측면 | SGF | CAS + Spatial CFG (Ours) |
|------|-----|--------------------------|
| **WHEN** | 고정 time window (이론 기반) | CAS score 기반 동적 결정 (per-step, per-prompt) |
| **WHERE** | Global (전체 latent) | Spatial mask 기반 선택적 적용 |
| **HOW** | MMD gradient repulsion (x_0 space) | Safe CFG: noise 방향 조정 (noise space) |
| **Reference data** | Unsafe image set D- 필요 | Anchor concept text만 필요 |
| **Concept specificity** | Data-driven, 암묵적 | Text-driven, 명시적 concept direction |

### 상세 비교

1. **WHEN 전략**
   - SGF: 모든 prompt에 동일한 time window 적용. 이론적으로 elegant하지만, safe prompt에서 불필요한 개입 발생
   - Ours (CAS): cos(d_prompt, d_target) > threshold일 때만 적용. Prompt-adaptive하여 COCO FP를 줄이는 데 유리

2. **WHERE 전략**
   - SGF: 완전히 부재. 전체 latent에 uniform repulsion
   - Ours: Cross-attention map 또는 noise/feature difference 기반 spatial mask. Unsafe 영역에만 선택적 개입하여 이미지 나머지 부분 보존

3. **HOW 전략**
   - SGF: 실제 unsafe 이미지 D-에 대한 MMD distance를 줄이는 data-driven 방식. 수학적으로 깔끔하지만 D- 구성에 의존
   - Ours: "nudity" concept direction (d_target)을 noise space에서 빼는 concept-driven 방식. Text만으로 concept 정의 가능

4. **Multi-concept 확장성**
   - SGF: concept별로 별도의 D-를 구성해야 함. 여러 concept 동시 erasing 시 D-를 합치면 inter-concept interference 가능
   - Ours: concept별 anchor text만 변경하면 됨. CAS threshold도 concept별로 독립 조정 가능. Phase 2 multi-concept 확장에 더 유리

5. **이론적 기여 활용 가능성**
   - SGF의 critical window 이론은 우리 방법에도 적용 가능한 insight 제공: CAS가 early step에서 더 자주 activate되는 것이 이론적으로도 타당함을 뒷받침
   - "Earlier is better" 원리는 CAS threshold 설계에서 early step에 더 민감하게 반응하도록 조정하는 근거가 될 수 있음

### 통합/영감 가능성

1. **SGF의 이론을 우리 방법에 적용**: CAS의 time-varying behavior에 대한 이론적 정당화로 활용 가능. "CAS가 early step에서 더 자주 activate되어야 하며, 이는 SGF의 critical window theorem과 일치한다"는 논리 구성 가능
2. **Hybrid 접근**: SGF의 MMD repulsion을 WHERE mask로 공간적으로 제한하면, global repulsion의 부작용을 줄이면서 이론적 장점을 유지할 수 있음
3. **비교 실험**: SGF는 동일한 데이터셋 (Ring-A-Bell, UnlearnDiff, MMA)을 사용하므로 직접 비교 가능. 특히 COCO FP 비교에서 우리 방법의 CAS가 갖는 장점을 부각할 수 있음
4. **NeurIPS 논문에서의 positioning**: SGF가 ICLR 2026이므로 concurrent/recent work으로 언급. "SGF는 WHEN에 대한 이론적 답변을 제공하지만 WHERE가 부재. 우리 방법은 WHEN(CAS) + WHERE(Spatial) + HOW(Safe CFG)의 세 차원 모두를 adaptive하게 결정한다"는 positioning이 가능

## 핵심 수식 요약

1. **MMD Potential**: E(x_t) = MMD^2_{k_sigma}({x_t}, D-)
2. **Repulsive Gradient**: nabla E = (2/sigma^2) Z(x_t) [x_t - sum w_i y_i]
3. **Guided ODE**: x_dot = f_theta(x_t, t) + lambda(t) nabla_x E(x_t)
4. **Critical Window Theorem**: guidance budget을 early step에 집중하면 safety bound가 최대화됨
5. **Practical Algorithm**: x'_{0|t} = x_hat_{0|t} + lambda(t) nabla E(x_hat_{0|t}, D-) (within time window C)
